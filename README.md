Time Series Analysis with Cryptocurrency

Project Description

This project focuses on predicting cryptocurrency prices using time series analysis techniques. Cryptocurrency markets are highly volatile and influenced by multiple factors, making accurate forecasting a challenging task.
The project analyses historical cryptocurrency price data and applies different machine learning and statistical models to forecast future price movements. Various preprocessing techniques, feature engineering methods, and forecasting models are used to improve prediction accuracy.

Technologies Used

Python
Pandas
NumPy
Matplotlib
TensorFlow
Prophet
Statsmodels
Scikit-learn

Features

Historical cryptocurrency price analysis
Data preprocessing and feature engineering
Volatility calculation using High–Low price difference
Time series forecasting using multiple models
Model performance evaluation using statistical metrics
Visualization of trends and predicted values
